## Cannibalization Types

**Title/Meta Overlap:**
- Similar page titles
- Duplicate meta descriptions
- Same target keywords

**Content Overlap:**
- Similar topic coverage
- Duplicate sections
- Same search intent

**Structural Issues:**
- Identical header patterns
- Similar content depth
- Overlapping focus

## Prevention Strategy

1. **Clear keyword mapping** - One primary keyword per page
2. **Distinct search intent** - Each page serves a different user need
3. **Unique angles** - Different perspectives on related topics
4. **Content consolidation** - Merge thin, overlapping pages
5. **Canonical tags** - Signal preferred version for near-duplicates

## Resolution Actions

| Issue | Action |
|---|---|
| Two pages targeting same keyword | Consolidate into one stronger page |
| Similar but distinct intent | Differentiate titles, meta, and content angles |
| Thin overlapping pages | Merge and redirect |
| Duplicate content | Use canonical tags or 301 redirects |
